# SMHRD 커뮤니티 (팀명 : Six_Month)


## 서비스 소개
* 서비스 명 : 리버스엔지니어링을 기반한 익명 커뮤니티 
* 서비스 설명 :
<br>
익명 커뮤니티를 통한 수강생 간 친목도모 및 취업정보 제공 

## 프로젝트 기간
2022.06.13 ~ 2022.06.18 (2주)
<br>

## 주요기능 
* 이메일 인증 기반 회원가입
* 익명 게시판 / 익명 댓글
* 게시판 검색 기능
* 취업 정보 제공 
* 일정표 제공 
<br>
